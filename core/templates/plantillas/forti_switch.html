Crear la community para snmp V1 y snmp V2:
****************************************

config system snmp community
    edit 1
    set events cpu-high ent-conf-change fan-detect intf-ip ip-conflict llv l2mac log-full mem-low psu-status sensor-alarm sensor-fault tkmem-hb-oo-sync
    config hosts
    edit 1
    set interface "internal"
    set ip 10.146.67.18 255.255.255.255
    next
    end
    set name "cirsa"
    next
end
